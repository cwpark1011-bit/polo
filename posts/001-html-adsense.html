코딩 초보가 HTML로 수익형 페이지 만드는 법

이 글은 GitHub Pages로 무료 배포 → 수익형 구조를 만드는 최소 단계를 정리합니다.

1) 저장소 만들기

  1. 프로젝트 페이지: 임의의 이름(예: polo)으로 새 저장소 생성
  2. index.html를 업로드하고, Settings → Pages에서 Branch main, Folder /(root) 저장
  3. 주소는 https://cwpark1011-bit.github.io/polo/

2) 기본 뼈대

<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width,initial-scale=1">
  <title>내 첫 수익형 페이지</title>
</head>
<body>
  <h1>안녕하세요!</h1>
  <div class="ad">광고 영역(승인 후 교체)</div>
</body>
</html>

3) 광고 영역 자리 만들어 두기

여기는 광고 자리(예시)

애드센스 승인 후 스크립트를 붙이되, 정책 위반(자체 클릭 유도, 과도한 배치 등)을 절대 하지 말 것.

4) 다음 단계

  • 공공데이터 시각화로 트래픽 늘리기
  • 애드센스 승인/배치 전략
